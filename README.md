# Chetak-alarm Chetak Connect

A lightweight EV charging time calculator and unplug reminder tool built with HTML, CSS, and JavaScript.

🔗 **Live Demo**: [View Demo](https://ashu-dev-stack.github.io/Chetak-Connect/)  
🔗 **GitHub Repository**: [Chetak-Connect](https://github.com/aSHU-dEv-stack/Chetak-Connect)

---

## 📋 Project Overview

This project is a frontend-based EV charging assistant that helps users estimate charging time and manage unplug reminders efficiently.

Chetak Connect demonstrates practical JavaScript logic building, DOM manipulation, and responsive UI structuring using pure HTML, CSS, and Vanilla JavaScript.

---

### 🎯 Features

- Battery percentage input system  
- Charging time estimation calculator  
- Time-to-full-charge display (Hours & Minutes)  
- Unplug reminder functionality  
- Clean and minimal user interface  
- Fully static and works offline  

---

### 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- GitHub Pages (for deployment)

## 🚀 Getting Started

### Local Development

1. Clone the repository:

```bash
git clone https://github.com/aSHU-dEv-stack/Chetak-Connect.git
```

2. Navigate into the project folder:

```bash
cd Chetak-Connect
```

3. Open `index.html` in your browser  
   or use the Live Server extension in VS Code.

## 💡 Key Components

- **Battery Input Section**: User enters current battery percentage  
- **Charging Calculator**: Computes estimated time required to reach 100%  
- **Time Display Module**: Displays charging time in structured format  
- **Unplug Reminder Logic**: Helps users avoid overcharging  
- **UI Layout**: Simple, user-friendly single-page structure
